About Sean

In the summer of 2017, I decided to make a career change. I was working as a music teacher and was ready for a new challange. Because I like to build things and enjoy the process of starting with an idea and transforming it into reality, software development sounded like a good fit. I enrolled in Galvanize's 6-month Web Development Immersive where I learned how to build full stack applications.

I am a web developer, educator, and musician.

Freelance Projects

Witkamp Guitar Lessons

Paul needed a website to help market his teaching business, collect new student information, and attract more students.

This is a single page React app connected to a Heroku server built with ReactJS, React Router, HTML5, CSS3, Flexbox, ES6, NodeJS, ExpressJS, and KnexJS.

School Projects

myTravels

myTravels is my capstone project for Galvanize's Web Development Immersive. The idea behind myTravels is to give users a travel oriented social platform to share photos, comments, and travel notes while connecting them to fellow travelers.

This project was built with HTML5, CSS3, ES6, ReactJS, React Router, NodeJS, ExpressJS, KnexJS, Cloundinary, and PostgreSQL. I followed the MVC design pattern to separate concerns.

Garden Walk

Garden Walk is a mobile React app that motivates people to walk more by rewarding users with an adorable garden they could grow with their steps.

This was a group project where I was partially responsible for building the API, RESTful routes, several React components, and planning/wireframing. Garden walk was built with HTML5, CSS3, JavaScript, ReactJS, React Router, NodeJS, ExpressJS, KnexJS, and PostgreSQL while following the MVC pattern.

Chord Genie

Chord Genie is a JavaScript program for the beginner guitarist who wants to learn to play and hear chords.

This was my first project at Galvanize. Chord Genie is built with JavaScript, HTML, and CSS.
